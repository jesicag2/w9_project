# React E-commerce Web Application

This project is a React-based e-commerce web application designed to provide a versatile platform for both customers and administrators to interact with.

## Features

- **Customer and CustomerAccount Management:** Create, read, update, and delete customer accounts.
- **Product Catalog:** Manage products by listing, adding, updating, and deleting them.
- **Order Processing:** Allow customers to place orders and retrieve order details.

## Technologies Used

- React.js
- React Router
- React-Bootstrap
- Axios
- Flask API (backend)
